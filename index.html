🦅 Добро пожаловать в Уни-карты

  • Вставьте конспект → «Собрать карту»
  • Перетаскивайте узлы, меняйте размер (8 хендлов по краям и углам)
  • Двойной клик по узлу — редактирование текста с форматированием B/I/U
  • Новое! Кнопка «📝 Текст» — создаёт свободный текстовый блок без фона
  • Новое! Кнопки A+/A- рядом с B/I/U меняют размер шрифта (8-128pt)
  • Ctrl/Cmd + / - для изменения размера шрифта, Ctrl+0 для сброса
  • Shift + выделение нескольких узлов
  • Shift при перетаскивании — отключает прилипание
  • Нажмите на пустой круг в палитре → выберите любой цвет → сохраните в палитру
  • Экспорт — кнопка 💻➡️ справа вверху
